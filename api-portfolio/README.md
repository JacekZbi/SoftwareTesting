##  How to Run

Follow these steps to import and run the automated test suite on your local machine:

### Prerequisites
* **Postman Desktop App** installed (or access to [Postman Web](https://www.postman.com/)).
---

### Step 1: Download the Project Files
Clone this repository or download the following two files from the repository folders directly to your computer:
1. `parabank-e2e-banking-workflow.postman_collection.json` (Located in `/postman_collections`)
2. `parabank-dev.postman_environment.json` (Located in `/postman_environments`)

### Step 2: Import Files into Postman
1. Open the Postman application.
2. Click the **Import** button located at the top-left of the sidebar.
3. Drag and drop both downloaded JSON files (the Collection and the Environment) into the import window.
4. Click **Import** to confirm.

### Step 3: Select the Environment
1. Click the dropdown and select **Para Bank QA**.
